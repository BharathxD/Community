# MicroCommunity Client

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

```bash
    npm i --save-dev @types
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Routes

| Routes           | Path                  |
| ---------------- | --------------------- |
| Homepage         | `/`                   |
| Login/Register   | `/auth`               |
| User Profile     | `/profile/[..userId]` |
